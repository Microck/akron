Overlay rulesets change presentation and proof output without replacing the primary ruleset.

## Streamer Mode

`Streamer Mode` hides local filesystem paths in Akron-owned UI and proof output. When active, Akron displays only the filename or final folder name instead of the full path.

Use it for streams, videos, screenshots, or public proof packages where local capture or export paths should not be exposed.

`Streamer Mode` is not a general privacy mode. It does not inspect every other mod, external overlay, visible game room data, or arbitrary text on screen.

## Proof-Mode

`Proof-mode` keeps proof surfaces compact and ruleset-aware. Akron normally enables it indirectly through `Submission Mode` or imported/community rulesets. The direct command path exists for QA and review automation, not as a primary player setting.

It does not:

- Certify legitimacy.
- Spoof completion.
- Change save stats.
- Replace moderator review.

## Low-Distraction State

The low-distraction state activates when all five visual-noise reduction channels are enabled: particles, trails, glitch, anxiety, and distortion. It is a derived presentation state, not a selectable ruleset or separate proof authority.

## When to Use These Overlays

| State | Use it when |
|---|---|
| Streamer Mode | Sharing screenshots, video, streams, or proof artifacts that should hide local filesystem paths. |
| Proof-mode | Submission Mode or community rulesets require compact proof metadata. |
| Low-distraction | Visual-noise settings already form a lower-noise setup across play sessions. |
