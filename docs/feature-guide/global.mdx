---
title: Global
description: Session-wide controls for speed, safety, submission metadata, autosave, and transition behavior, including FPS and TPS bypass when Motion Smoothing is loaded.
---

The Global tab collects options that affect the whole current Akron session rather than one HUD surface or one room tool.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| FPS Bypass | Numeric toggle | Raise the frame draw cadence while keeping the configured simulation cadence. Only visible when the Motion Smoothing mod is loaded. | Cheat |
| TPS Bypass | Numeric toggle | Change the simulation tick cadence. Only visible when the Motion Smoothing mod is loaded. | Cheat |
| Timescale | Number | Adjust the gameplay timescale multiplier between 0.1× and 2.0×. | Cheat |
| Safe Mode | Toggle | Safe Mode indicator plus optional local stat guards for deaths, jumps, and best-run fields. | Normal clear |
| Submission Mode | Toggle | Enable recording metadata defaults and related warning surfaces without changing gameplay by itself. | Goldberry/Hardlist clear |
| Freeze Attempts | Toggle | Restore the current save-slot death counter while Safe Mode is active. | Cheat |
| Pause Buffering | Toggle | Choose whether configured hotkeys can fire while Celeste is paused. | Cheat |
| Control Display | Toggle | Display live inputs on a customizable board with editable keys, bindings, colors, and placement. | Goldberry/Hardlist clear |
| Autosave | Toggle | Save Akron-owned backup state automatically at configured safe points. | Normal clear |
| Transition Speed | Numeric toggle | Adjust the next room-transition duration multiplier. | Cheat |
