---
title: Global
description: Session-wide controls for speed, safety, submission metadata, autosave, and transition behavior.
---

The Global tab collects options that affect the whole current Akron session rather than one HUD surface or one room tool.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Timescale | Number | Open speed controls for decreasing or increasing gameplay timescale. | Cheat |
| Safe Mode | Toggle | Safe Mode indicator plus optional local stat guards for deaths, jumps, and best-run fields. | Regular clean |
| Submission Mode | Toggle | Enable recording metadata defaults and related warning surfaces without changing gameplay by itself. | Goldberry/Hardlist clean |
| Freeze Attempts | Toggle | Restore the current save slot's death counter while Safe Mode is active. | Cheat |
| Pause Buffering | Toggle | Choose whether configured hotkeys can fire while Celeste is paused. | Cheat |
| Control Display | Toggle | Display live inputs on a customizable board with editable keys, bindings, colors, and placement. | Goldberry/Hardlist clean |
| Autosave | Toggle | Save Akron-owned backup state automatically at configured safe points. | Regular clean |
| Transition Speed | Numeric toggle | Adjust the next room-transition duration multiplier. | Cheat |
