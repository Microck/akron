---
title: Global
description: Session-wide controls for speed, safety, submission metadata, autosave, and transition behavior.
---

The Global tab collects options that affect the entire Akron session rather than one HUD surface or one room tool.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Timescale | Numeric toggle | Enable a configured gameplay timescale and edit its multiplier independently. | Cheat |
| Transition Speed | Numeric toggle | Adjust the next room-transition speed multiplier. Values below 1x are slower and values above 1x are faster. | Cheat |
| Frame Stepper | Toggle | Step one frozen frame and configure hold-to-repeat stepping. | Cheat |
| FPS Bypass | Numeric toggle | Raise the draw cadence or enable Motion Smoothing's smoothing pipeline while keeping Celeste physics at 60 FPS. | Regular clean |
| TPS Bypass | Numeric toggle | Change the simulation tick cadence. | Cheat |
| Safe Mode | Toggle | Safe Mode indicator plus optional local stat guards for deaths, jumps, and best-run fields. | Regular clean |
| Freeze Attempts | Toggle | Restore the current save slot's death counter while Safe Mode is active. | Cheat |
| Submission Mode | Toggle | Enable recording metadata defaults and related warning surfaces without changing gameplay itself. | Goldberry/Hardlist clean |
| Pause Buffering | Toggle | Choose whether configured hotkeys can fire while Celeste is paused. | Cheat |
| Autosave | Toggle | Save Akron-owned backup state automatically at configured safe points. | Regular clean |

## Option menus

Rows with a triangle expose the controls listed below.

| Option | Menu control | What it changes | Attempt status |

| FPS Bypass | Method | Interval keeps render cadence aligned to physics ticks. Dynamic allows arbitrary FPS but is riskier for code that hooks the main tick. | Regular clean |
| FPS Bypass | Smooth Camera | Fancy is highest quality, Fast is cheaper but can jitter backgrounds, Off leaves the camera pixel-locked. | Regular clean |
| FPS Bypass | Objects | Extrapolate predicts between physics frames. Interpolate uses prior frames and can add 1-2 frames of delay. | Regular clean for Extrapolate; Cheat for Interpolate |

| FPS Bypass | Smooth background | Reference option for high-resolution background compositing in Fancy camera smoothing. | Regular clean |
| FPS Bypass | Smooth foreground | Reference option for high-resolution foreground compositing in Fancy camera smoothing. | Regular clean |
