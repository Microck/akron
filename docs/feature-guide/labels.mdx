---
title: Labels
description: HUD label visibility, room labels, status labels, input panels, counters, and custom labels.
---

The Labels tab controls label-style HUD surfaces. It includes a master visibility gate plus individual rows that keep their own enabled state.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Visible | Toggle | Master visibility gate for label-style HUD surfaces. Individual label options keep their enabled state while this is off. | Regular clean |
| Player Overlap | Toggle | Fade or move HUD labels when Madeline overlaps them. | Regular clean |
| Death Stats | Toggle | Show a configurable death counter in the Akron HUD with visibility controls. | Goldberry/Hardlist clean |
| Room | Toggle | Show the current map and room label in the Akron HUD. | Regular clean |
| Status | Toggle | Show compact HUD labels for presentation overlays and attempt status. | Regular clean |
| Toasts | Toggle | Show option feedback messages as plain label-style text instead of a centered background panel. | Regular clean |
| Cheat Indicator | Toggle | Configure the dedicated attempt-status indicator alignment, scale, opacity, and only-flagged behavior. | Regular clean |
| Input History | Toggle | Show the current input chord and an optional plain-text input history panel. | Goldberry/Hardlist clean |
| Inputs per second | Toggle | Display rolling input presses per second, total presses, and max rate. | Regular clean |
| Dash Stats | Toggle | Show dash-count statistics in the label system. | Regular clean |
| Jump Stats | Toggle | Show jump-count statistics in the label system. | Regular clean |
| StartPos HUD | Toggle | Show, hide, and style the HUD label that displays the active StartPos index. | Cheat |
| Room Timer | Toggle | Show current room timing information in the Akron HUD. | Regular clean |
| Room Stat Tracker | Toggle | Show configurable room name, deaths, timer, berry count, and alive-time lines in the label HUD. | Regular clean |
| Attempts | Toggle | Show total attempt count in the label system. | Goldberry/Hardlist clean |
| No Short Numbers | Toggle | Show full HUD numbers instead of compact K/M formatting. | Goldberry/Hardlist clean |
| + Custom | Action | Add a new custom HUD label row. | Regular clean |
