## Option Menus

Rows with a triangle expose the controls listed below.

| Option | Menu control | What it changes | Status |
|---|---|---|---|
| Audio Splitter | Reload Devices | Refresh FMOD output devices visible to Celeste. | Regular Clean |
| Audio Splitter | Main | Choose the output device for the main audio route. | Regular Clean |
| Audio Splitter | Music | Choose the output device for the music audio route. | Regular Clean |
| Audio Splitter | SFX | Choose the output device for the sound-effects audio route. | Regular Clean |
| Allow Low Volume | Music | Set the forced low music volume level. | Regular Clean |
| Allow Low Volume | SFX | Set the forced low SFX volume level. | Regular Clean |
| Audio Speed | Policy: Normal, Sync Timescale, Independent | Choose normal speed, Akron timescale sync, or an independent audio speed. | Normal clear |
| Audio Speed | Speed | Set the independent audio speed multiplier. | Regular Clean |
| Pitch Shift | Policy: Preserve, Follow Speed, Independent | Choose normal pitch, pitch following audio speed, or an independent pitch. | Normal clear |
| Pitch Shift | Pitch | Set the independent pitch multiplier. | Regular Clean |
| Per-sound row | Volume | Set volume percentage (0–200) for the matching sound event. Only applies while the sound row is enabled. | Regular Clean |
