## Option menus

Rows with a triangle expose the controls listed below.

| Option | Menu control | What it changes | Attempt status |
|---|---|---|---|
| Audio Splitter | Reload Devices | Refresh FMOD output devices visible to Celeste. | Normal clear |
| Audio Splitter | Main | Choose the output device for the main audio route. | Normal clear |
| Audio Splitter | Music | Choose the output device for the music audio route. | Normal clear |
| Audio Splitter | SFX | Choose the output device for the sound effects audio route. | Normal clear |
| Allow Low Volume | Music | Set forced low music volume. | Normal clear |
| Allow Low Volume | SFX | Set forced low SFX volume. | Normal clear |
| Audio Speed | Policy: Normal/SyncTimescale/Independent | Choose normal speed, Akron timescale sync, or an independent audio speed. | Normal clear |
| Audio Speed | Speed | Set the independent audio speed multiplier. | Normal clear |
| Pitch Shift | Policy: Preserve/FollowSpeed/Independent | Choose normal pitch, pitch following audio speed, or an independent pitch. | Normal clear |
| Pitch Shift | Pitch | Set the independent pitch multiplier. | Normal clear |
| [Sound event] | Volume | Set volume percentage for the overridden sound (0% to 200%). | Normal clear |
