---
title: External Tools
description: Options that appear only when Motion Smoothing, Speedrun Tool, CelesteTAS, or Extended Variant Mode is loaded.
---

External tool rows are not part of Akron's base overlay tabs. They appear only when the relevant mod is loaded, and they are documented separately so the main feature pages stay focused on Akron's own options.

Motion Smoothing adds FPS Bypass and TPS Bypass rows to the Global tab. Extended Variant Mode can also expose variant-specific rows at runtime; those dynamic rows depend on the installed Extended Variant Mode option list.

| Tool | Option | Control | What it does | Attempt status |
|---|---|---|---|---|
| Motion Smoothing | FPS Bypass | Numeric toggle | Raise the draw cadence while keeping the configured simulation cadence when possible. | Cheat |
| Motion Smoothing | TPS Bypass | Numeric toggle | Change the simulation tick cadence. | Cheat |
| Speedrun Tool | SRT Status | Action | Show whether Akron can read Speedrun Tool status, slot state, and room timer right now. | Normal clear |
| Speedrun Tool | SRT Slot | Action | Show or cycle the active Akron slot used for Speedrun Tool state capture and restore actions. | Normal clear |
| Speedrun Tool | SRT Capture State | Action | Capture the active Speedrun Tool state slot. | Cheat |
| Speedrun Tool | SRT Restore State | Action | Restore the active Speedrun Tool state slot. | Cheat |
| Speedrun Tool | SRT Clear State | Action | Clear the active Speedrun Tool state slot. | Cheat |
| Speedrun Tool | SRT Room Time | Action | Show Speedrun Tool's current room timer value when its room-timer export is available. | Normal clear |
| Speedrun Tool | Export Room Times | Action | Export room-time data when the optional room timer source is available. | Normal clear |
| CelesteTAS | TAS Status | Action | Show whether CelesteTAS reports an active or currently running TAS. | Normal clear |
| CelesteTAS | Configured TAS File | Action | Show the configured TAS file basename used by Akron's CelesteTAS handoff. | Normal clear |
| CelesteTAS | Play Configured TAS | Action | Start CelesteTAS playback for Akron's configured TAS file path. | Cheat |
| Extended Variant Mode | Reset Extended | Action | Reset all non-vanilla external variant settings to their defaults. | Normal clear |
| Extended Variant Mode | Reset Vanilla | Action | Reset external vanilla variant wrappers to their defaults. | Normal clear |
| Extended Variant Mode | Extended Variants Master | Toggle | Toggle Extended Variant Mode's master switch when the mod is loaded. | Cheat |
| Extended Variant Mode | Extended Variants Randomizer | Toggle | Toggle and configure Extended Variant Mode's randomizer when the mod is loaded. | Cheat |

## Option Menus

Rows with a triangle expose the controls listed below.

| Option | Menu control | What it changes | Status |
|---|---|---|---|
| FPS Bypass | Target FPS | Set the draw target while FPS Bypass is enabled. | Cheat |
| FPS Bypass | Method | Choose interval or dynamic render cadence behavior. | Cheat |
| FPS Bypass | Smooth Camera | Choose Fancy, Fast, or Off camera smoothing. | Cheat |
| FPS Bypass | Objects | Choose extrapolate or interpolate object smoothing. | Cheat |
| FPS Bypass | TAS mode | Keep overworld updates locked for TAS compatibility. | Cheat |
| FPS Bypass | Subpixel Madeline | Draw Madeline and held items at subpixel render positions. | Cheat |
| FPS Bypass | Smooth background | Use high-resolution background compositing in Fancy camera smoothing. | Cheat |
| FPS Bypass | Smooth foreground | Use high-resolution foreground compositing in Fancy camera smoothing. | Cheat |
| FPS Bypass | Hide edge gaps | Slightly zoom the level to hide gaps introduced by fractional camera offsets. | Cheat |
| FPS Bypass | Silly mode | Apply an experimental smoothing preset to FPS Bypass rendering. | Cheat |
| TPS Bypass | Target TPS | Set the simulation update target while TPS Bypass is enabled. | Cheat |
| Extended Variants Randomizer | Change variants randomly | Enable the external variant randomizer loop. | Cheat |
| Extended Variants Randomizer | Reroll mode | Replace the randomized variant set instead of changing one option at a time. | Cheat |
| Extended Variants Randomizer | Display enabled variants | Show the active variant list on-screen. | Cheat |
| Extended Variants Randomizer | Interval | Set seconds between randomizer changes; zero uses screen-change behavior. | Cheat |
| Extended Variants Randomizer | Max | Set the maximum number of simultaneously randomized variants. | Cheat |
