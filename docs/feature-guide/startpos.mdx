title: StartPos
description: Capture, restore, place, cycle, and respawn from Akron StartPos slots.

The StartPos tab is Akron's player position and room-state restore workflow. These rows are meant for room-lab routing and are marked as Cheat when used in submitted clean attempts.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Smart StartPos | Toggle | Choose a nearby stored StartPos automatically when respawning. | Cheat |
| StartPos Slot | Selector | Select the active StartPos slot. | Cheat |
| Respawn at StartPos | Toggle | Respawn at the selected StartPos instead of the vanilla room spawn. | Cheat |
| StartPos | Action | Set, load, or clear the active full-state snapshot. Advanced options control placement, cycling, respawn behavior, and spawn config. | Cheat |
| Place StartPos | Toggle | Toggle cursor placement for StartPos. The placement popup owns spawn configuration. | Cheat |
| StartPos Switcher | Action | Configure and use previous/next StartPos switching. | Cheat |
