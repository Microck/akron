title: StartPos
description: Capture, restore, place, cycle, and respawn from Akron StartPos slots for room-lab routing and practice.
---

The StartPos tab provides Akron's position and room-state snapshot workflow. These rows are designed for room-lab routing practice and count as **Cheat** in submitted clean attempts.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| StartPos | Action | Set, load, or clear the active full-state snapshot. The popup exposes placement, cycling, respawn, and spawn-config controls. | Cheat |
| Place StartPos | Action | Toggle cursor placement for StartPos. The placement popup controls spawn configuration. | Cheat |
| Smart StartPos | Toggle | Automatically choose a nearby stored StartPos when respawning. | Cheat |
| StartPos Switcher | Action | Cycle between saved StartPos entries using Previous/Next, with optional menu bindings. | Cheat |
| StartPos Slot | Selector | Select the active StartPos slot from the configured slot count. | Cheat |
| Respawn at StartPos | Toggle | Respawn at the selected StartPos instead of the vanilla room spawn. | Cheat |

## Option menus

Rows with a triangle indicator expose the controls listed below.

| Option | Menu control | What it changes | Status |
|---|---|---|---|

| StartPos | Previous | Cycle to the previous saved StartPos in chapter order. | Cheat |

| Place StartPos | Open editor | Freeze gameplay, activate the free camera, and place StartPos previews with the mouse. | Cheat |

| Place StartPos | Dashes | Keep the native dash count (−1) or force 0–5 dashes after spawning. | Cheat |
