title: Player
description: Player movement, resource, presentation, assist, invincibility, noclip, trail, and comfort controls in Akron.
---

The Player tab covers Madeline-facing controls. Visual-only rows are generally presentation settings; movement, resource, teleport, and input-assist rows are Cheat.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Air Jumps | Toggle | Open controls for extra airborne jumps. | Cheat |
| Always Show Trail | Toggle | Force Madeline's dash trail to render continuously. | Normal clear |
| Click Teleport | Toggle | Hold the cursor hotkey, then click to teleport to the in-room cursor position. | Cheat |
| Custom Trail | Toggle | Customize forced trail color, rainbow mode, pulse, opacity, and cut rate. | Normal clear |
| Dash Bar | Toggle | Show a dash-resource display. | Cheat |
| Dash Count | Toggle | Override the active dash count used for room entry, transitions, and dash refills. | Cheat |
| Dash Number | Toggle | Show the current dash count as a number. | Cheat |
| Fast Lookout | Numeric toggle | Speed up vanilla lookout camera movement while holding the configured bind. | Cheat |
| Frame Stepper | Toggle | Step one frozen frame and configure hold-to-repeat stepping. | Cheat |
| Golden Start | Action | Run Celeste's first-room golden-start helper. | Goldberry/Hardlist clear |
| Golden Transparency | Toggle | Make the golden berry and follower more transparent while the option is enabled. | Normal clear |
| Grab Mode | Toggle | Open grab-mode controls for Hold, Toggle, and Invert. | Normal clear |
| Ground Refills | Toggle | Change whether standing on the ground restores dash and stamina resources. | Cheat |
| Hazard Accuracy | Toggle | Prevent deaths while tracking invalid hazard contacts, accuracy, and optional tint. | Cheat |
| Hide Player | Toggle | Hide Madeline while keeping gameplay state unchanged. | Normal clear |
| Infinite Dash | Toggle | Replenish dash count after dash consumption during gameplay updates. | Cheat |
| Infinite Stamina | Toggle | Keep player stamina replenished during gameplay updates. | Cheat |
| Invincibility | Toggle | Use Celeste's native Assist invincibility behavior and rescue bottomless falls back to the room spawn. | Cheat |
| Madeline Colors | Toggle | Customize Madeline's dash-state hair colors without flattening every state to one color. | Normal clear |
| Madeline Hair Length | Toggle | Customize Madeline's normal dash-state hair segment counts. | Normal clear |
| Madeline Effect Sync | Toggle | Match selected player visual effects to Madeline's active hair color. | Normal clear |
| No Death Effect | Toggle | Suppress death particles and death-effect visual noise. | Normal clear |
| No Ghost Trail | Toggle | Suppress Celeste player trail ghosts. | Normal clear |
| No Stamina Flash | Toggle | Suppress the low-stamina flash effect while leaving stamina rules unchanged. | Normal clear |
| No Trails | Toggle | Clear player trail effects while enabled. | Normal clear |
| No Respawn Animation | Toggle | Shorten and hide the death-body respawn animation. | Normal clear |
| Noclip | Toggle | Open fly-noclip controls for movement speed, grab-held slow speed, and draw-on-top visibility. | Cheat |
| Prevent Down Dash Redirects | Toggle | Keep downward dash input from being converted to horizontal-only redirects. | Cheat |
| Show Trajectory | Toggle | Draw a short current-velocity movement preview. | Cheat |
| Speed Number | Toggle | Show current speed as a small number above Madeline. | Cheat |
| Stamina Bar | Toggle | Show a stamina-resource display. | Cheat |
