---
title: Shortcuts
description: One-shot actions for options, retry, reloads, input-assist shortcuts, and cutscene skipping.
---

The Shortcuts tab exposes action rows that are useful during play. Reloads and input-assist actions have different policy effects, so check the Attempt status column before using them in a submission context.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Open Options | Action | Jump to Akron options and appearance controls inside the overlay. | Goldberry/Hardlist clean |
| Retry | Action | Restart the current room through Celeste's retry flow. | Regular clean |
| Restart Level | Action | Restart the current chapter through Celeste's loader. | Regular clean |
| Reload Room | Action | Restart the current room. | Cheat |
| Reload Chapter | Action | Restart the current chapter through Celeste's loader. | Regular clean |
| Neutral Drop | Action | Throw the held object downward via a dedicated shortcut. | Cheat |
| Backboost | Action | Flip facing and throw the held object via a dedicated shortcut. | Cheat |
| Skip Cutscene / Dialogue | Action | Request to skip the active cutscene or dialogue sequence. | Regular clean |
