---
title: Shortcuts
description: One-shot actions for options, retry, reloads, input-assist shortcuts, and cutscene skipping.
---

The Shortcuts tab exposes action rows that are useful during play. Reloads and input-assist actions have different policy effects, so check the status column before using them in a submission context.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Open Options | Action | Jump to Akron option and appearance controls inside the overlay. | Goldberry/Hardlist clear |
| Retry | Action | Restart the current room through Celeste's retry flow. | Normal clear |
| Restart Level | Action | Restart the current chapter through Celeste's loader. | Normal clear |
| Reload Room | Action | Restart the current room. | Cheat |
| Reload Chapter | Action | Restart the current chapter through Celeste's loader. | Normal clear |
| Neutral Drop | Action | Throw the held object downward from a dedicated shortcut. | Cheat |
| Backboost | Action | Flip facing and throw the held object from a dedicated shortcut. | Cheat |
| Skip Cutscene / Dialogue | Action | Request the active cutscene or dialogue sequence to skip. | Normal clear |
