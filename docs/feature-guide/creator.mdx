title: Creator
description: Map inspection, camera, capture, room warp, and export tools for creators and QA.
---

The Creator tab is for map inspection and documentation work. Warps, independent camera tools, entity data, and hitbox-style inspection can mark attempts as Cheat.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Free Camera | Toggle | Pan the camera freely inside the current room while Madeline is not player-controlled. | Cheat |
| Cursor Tools | Toggle | Temporarily enable selected cursor-driven tools while holding one cursor-tools bind. Its left-click action can be Click Teleport or Entity Inspector. | Cheat |
| Cursor Zoom | Toggle | Hold the cursor bind and use the mouse wheel to magnify the area under the cursor. | Cheat |
| Entity Inspector | Toggle | Enable cursor-held pinning for entities or triggers, highlighting hovered and pinned targets, and reading runtime and map-source details. | Cheat |
| Room Capture | Action | Capture overlapping tiles across the current room. | Goldberry/Hardlist clean |
| Map Capture | Action | Capture each room in the current map. | Goldberry/Hardlist clean |
| Previous Room | Action | Move the selected map room cursor to the previous available room. | Regular clean |
| Next Room | Action | Move the selected map room cursor to the next available room. | Regular clean |
| Warp Selected Room | Action | Warp to the currently selected room. | Cheat |
| Warp To Previous In Order | Action | Warp to the previous room in map order. | Cheat |
| Warp To Next In Order | Action | Warp to the next room in map order. | Cheat |
| Open Debug Map | Action | Open Celeste's debug map scene. | Cheat |
| Camera Offset | Toggle | Toggle a configured current-level camera offset for map inspection. | Cheat |
| Export Room Stats | Action | Write Akron's current stored room stats for this map to a TSV file. | Regular clean |
