---
title: Bypass
description: Save and completion mutation tools for chapter unlocks, completion state, berry collection, and path access in the Bypass tab.
---

The Bypass tab provides save-mutation and completion-state tools. Use these for local testing and recovery, not for clean-run setup.

:::info
FPS Bypass and TPS Bypass appear in the **Global** tab when the [Motion Smoothing](/docs/feature-guide/external-tools) mod is loaded. Their popup controls (target FPS/TPS, method, camera smoothing, object smoothing, TAS mode, and other options) live alongside Global-tab entries rather than here.
:::

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Instant Complete | Action | Route the current chapter through Celeste's completion flow immediately. | Cheat |
| Uncomplete Level | Action | Clear current chapter completion fields in save data. | Cheat |
| Unlock A-Sides | Action | Unlock main chapter access through Celeste's native UnlockedAreas field. | Cheat |
| Unlock B-Sides | Action | Mark cassette unlocks for chapters with B-Sides. | Cheat |
| Unlock C-Sides | Action | Enable Celeste's native C-Side visibility path. | Cheat |
| Unlock All Levels | Action | Apply Celeste's native unlock-all fields. | Cheat |
| Unlock Golden Berries | Action | Expose golden berry availability without marking berries collected. | Cheat |
| Unlock Paths | Action | Enable native path, checkpoint, side, and heart-gate bypasses. | Cheat |
| Obtain Room Berries | Action | Mark selected berry types in the current room as obtained. | Cheat |
| Obtain Chapter Berries | Action | Mark selected berry types in the current chapter side as obtained. | Cheat |
| Berry Obtain Options | Action | Choose whether obtain actions include regular strawberries, golden berries, and the moon berry. | Normal clear |

## Option Menus

Rows with a triangle indicator expose the controls listed below.

| Option | Menu control | What it changes | Status |
|---|---|---|---|
| Berry Obtain Options | Regular strawberries | Include normal red strawberries and winged red strawberries in obtain actions. | Normal clear |
| Berry Obtain Options | Golden berries | Include goldenBerry entities and 1A's winged golden memorial berry in obtain actions. | Normal clear |
| Berry Obtain Options | Moon berry | Include Farewell's moon berry when it exists in the selected room or chapter. | Normal clear |
