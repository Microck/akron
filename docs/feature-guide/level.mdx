---
title: Level
description: Level-scoped tools for automation, freezing, proof helpers, visual suppression, and hitbox display.
---

The Level tab focuses on the current level scene. Several rows are useful for routing and review, but state-changing rows can mark the attempt as Cheat.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Auto Kill | Toggle | Automatically kill the player after a configured map-time threshold or inside selected regions. | Cheat |
| Auto Deafen | Toggle | Press the configured Discord Toggle Deafen hotkey after entering a selected region until death or reset. | Regular clean |
| Confirm Restart | Toggle | Require a confirmation prompt before restarting the current room attempt. | Goldberry/Hardlist clean |
| Confirm Full Reset | Toggle | Require a confirmation prompt before fully reloading the current chapter. | Goldberry/Hardlist clean |
| Freeze Gameplay | Action | Freeze gameplay updates until toggled off or advanced by the frame stepper. | Cheat |
| Respawn Time | Numeric toggle | Open respawn timing controls, including whether speedhack affects the delay. | Regular clean |
| Pause Timer | Numeric toggle | Delay resumed gameplay after unpausing and show a center countdown. | Cheat |
| Pause Tracker | Toggle | Record pause count, rapid pauses, and paused time. | Goldberry/Hardlist clean |
| Lag Pauser | Toggle | Pause after a detected frame-time spike. | Goldberry/Hardlist clean |
| Freeze Timer While Paused | Toggle | Halt level and journal timer accumulation while paused. | Cheat |
| Hide Pause Menu | Toggle | Hide Celeste's pause menu while the game remains paused. | Regular clean |
| Light Level | Toggle | Adjust light presentation for the current level. | Regular clean |
| Bloom Level | Toggle | Adjust bloom presentation for the current level. | Regular clean |
| Screen Tint | Toggle | Apply a configurable screen tint. | Regular clean |
| Screenshake | Toggle | Override native Celeste screenshake. Intensity 0 disables shake. | Goldberry/Hardlist clean |
| Refill Clarity | Toggle | Highlight already-visible one-use refills with a configurable outline. | Regular clean |
| Deload Spinners | Toggle | Simulate spinner deload timing while the option is enabled. | Cheat |
| Skip Intro | Toggle | Hold confirm on repeated B-side intro title screens to enter the map. | Regular clean |
| Skip Postcards | Toggle | Hold confirm on level-entry postcards to jump straight into the map. | Regular clean |
| Reduced Visual Noise | Toggle | Apply Akron's reduced visual-noise presentation settings. | Regular clean |
| No Particles | Toggle | Suppress particle presentation. | Regular clean |
| No Glitch | Toggle | Suppress glitch-style visual effects. | Regular clean |
| No Anxiety | Toggle | Suppress anxiety-style visual effects. | Regular clean |
| No Distortion | Toggle | Suppress distortion-style visual effects. | Regular clean |
| Hide Snow | Toggle | Hide snow presentation. | Regular clean |
| Hide Wind Snow | Toggle | Hide wind-snow presentation. | Regular clean |
| Hide Waterfalls | Toggle | Hide waterfall presentation. | Regular clean |
| Hide Tentacles | Toggle | Hide tentacle presentation. | Regular clean |
| Hide Heat Distortion | Toggle | Hide heat-distortion presentation. | Regular clean |
| No Death Wipe | Toggle | Suppress black screen wipes while preserving completion callbacks when configured. | Regular clean |
| No Freeze Frames | Toggle | Suppress native hitstop/freeze-frame pauses. This changes gameplay timing. | Cheat |
| Show Hitboxes | Toggle | Open live hitbox controls for collision boxes, filters, colors, opacity, thickness, and style. | Cheat |
| Fix Hitbox Pixels | Toggle | Render circular hitboxes as sampled collision pixels instead of a smooth approximate outline. | Regular clean |
| Show Hitbox Trail | Toggle | Draw recent player collision boxes behind Madeline. | Cheat |
| Show Hitboxes On Death | Toggle | Show the death-object hitbox after dying without enabling live hitboxes. | Regular clean |
| Show Triggers | Toggle | Draw invisible Celeste trigger regions without turning on all hitboxes. | Cheat |

## Option menus

Rows with a triangle expose the controls listed below.

| Option | Menu control | What it changes | Attempt status |
|---|---|---|---|
| Auto Kill | Map-time threshold | Kill the attempt when current map time reaches the configured threshold. | Cheat |
| Auto Kill | Selected areas | Kill the attempt when Madeline enters selected rectangles. | Cheat |
| Auto Kill | Draw areas | Draw selected Auto Kill rectangles below the Akron menu. | Cheat |
| Auto Kill | Clear areas | Remove all selected Auto Kill rectangles. | Cheat |
| Auto Deafen | Hotkey | Set the Discord Toggle Deafen hotkey Akron will press. | Regular clean |
| Auto Deafen | Test hotkey | Press the configured hotkey immediately. | Regular clean |
| Auto Deafen | Selected areas | Trigger deafen when Madeline enters selected rectangles. | Regular clean |
| Auto Deafen | Draw areas | Draw selected Auto Deafen rectangles below the Akron menu. | Regular clean |
| Auto Deafen | Clear areas | Remove all selected Auto Deafen rectangles and restore if Akron believes Discord is deafened. | Regular clean |
| Respawn Time | Seconds | Configure the post-death respawn delay. | Regular clean |
| Respawn Time | Real time | Use real elapsed time so Akron timescale does not stretch or shrink the delay. | Regular clean |
| Pause Timer | Seconds | Configure how long gameplay is held after unpausing. | Cheat |
| Pause Timer | Remove pause darkening | Remove Celeste's leftover pause darkening during the countdown. | Cheat |
| Lag Pauser | Threshold ms | Set the frame-time spike threshold that triggers the pause. | Goldberry/Hardlist clean |
| Lag Pauser | Triggers | Show how many frame-time spikes triggered Lag Pauser. | Goldberry/Hardlist clean |
| Lag Pauser | Last spike | Show the duration of the most recent frame-time spike. | Goldberry/Hardlist clean |
| Lag Pauser | Reset threshold | Restore the default spike threshold. | Goldberry/Hardlist clean |
| Light Level | Percent | Set the lighting override percentage while Light Level is enabled. | Regular clean |
| Bloom Level | Percent | Set bloom suppression or amplification while Bloom Level is enabled. | Regular clean |
| Screen Tint | Opacity | Set screen tint opacity. | Regular clean |
| Screen Tint | Tint | Choose the screen tint color. | Regular clean |
| Deload Spinners | Seconds before deload | Configure spinner deload simulation timing. | Cheat |
| Show Hitboxes | Active only | Hide inactive entities from live hitbox rendering. | Cheat |
| Show Hitboxes | Hide player | Hide the player collider from live hitbox rendering. | Cheat |
| Show Hitboxes | Hazards | Draw spike, blade, and death-object hitboxes. | Cheat |
| Show Hitboxes | Solids | Draw solid collision boxes. | Cheat |
| Show Hitboxes | Triggers | Draw trigger areas. | Cheat |
| Show Hitboxes | Sync Hitboxes | Clear cached hitbox draw state and rebuild live hitboxes on the next frame. | Cheat |
| Show Hitboxes | Show Hitbox Trail | Draw recent player collision boxes behind Madeline. | Cheat |
| Show Hitboxes | Length | Set how many previous player hitbox positions are kept. | Cheat |
| Show Hitboxes | Trail % | Set maximum opacity for older hitbox trail samples. | Cheat |
| Show Hitboxes | Line | Set live hitbox outline thickness. | Cheat |
| Show Hitboxes | Black outline | Add a black contrast border behind colored hitbox lines. | Cheat |
| Show Hitboxes | Fill % | Set interior opacity for drawn hitboxes. | Cheat |
| Show Hitboxes | Player | Set Madeline/player hitbox color. | Cheat |
| Show Hitboxes | Solids color | Set solid collision grid color. | Cheat |
| Show Hitboxes | Hazards color | Set spike, blade, and death-object color. | Cheat |
| Show Hitboxes | Triggers color | Set trigger area color. | Cheat |
| Show Hitboxes | Other | Set fallback color for other collidable entities. | Cheat |
| Show Hitboxes | Reset style | Restore Akron's default hitbox style. | Cheat |
| Show Hitbox Trail | Length | Set how many recent player collision boxes are kept. | Cheat |
| Show Hitbox Trail | Opacity | Set hitbox trail opacity. | Cheat |
| Show Hitboxes On Death | All hitboxes | During the death window, draw the regular hitbox set using Show Hitboxes settings. | Regular clean |
| Show Hitboxes On Death | Player marker | Mark Madeline's recorded death position even when all hitboxes or Hide player is on. | Regular clean |
| Show Hitboxes On Death | Death object | Set the last recorded death-object hitbox color. | Regular clean |
| Show Hitboxes On Death | Player marker color | Set the last recorded player-position marker color. | Regular clean |
