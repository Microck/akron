The Interface tab controls the Akron overlay itself and the public `.akr` pack workflows.

| Option | Control | What it does | Attempt status |
|---|---|---|---|
| Theme | Selector | Select the Akron overlay color theme. | Regular clean |
| UI Scale | Number | Scale Akron overlay text and controls. | Regular clean |
| Opacity | Number | Set Akron overlay background opacity. | Regular clean |
| Streamer Mode | Toggle | Show filenames instead of full local filesystem paths. | Regular clean |
| Pause While Open | Toggle | When enabled, opening the Akron menu stops gameplay updates until the menu closes. | Regular clean |
| Export Profile | Action | Export the selected Whole, StartPos, Keybinds, Auto Kill, Auto Deafen, Recorder, Audio, or HUD profile scope as a `.akr` profile pack. | Regular clean |
| Import Profile | Action | Open a file browser to import a selected `.akr` profile pack using the selected profile scope. | Regular clean |
| Community Packs | Action | Search map-linked community `.akr` packs from the configured community pack index. | Regular clean |
| Search Autofocus | Toggle | Focus Akron search automatically when the overlay opens. | Regular clean |
| Search | Text input | Filter visible Akron rows by label or search alias. | Regular clean |
